Import World

#1 Import the world into your Minecraft folder

When importing the world in single player, only you or the user on your local device can play. To play the world with other players, see below.
To be able to play the world in single player, you must follow out these steps after downloading:
#1
Right click on the .zip file "Projekt City [V5.0]" and click "extract all"
#2
Click on the bottom right on "extract"
#3
Wait for the file to extract and return to your previous directory
#4
Right click on the extracted file and copy it
#5
Press the windows symbol and type "run" and open the app, enter "%appdata%" and click "Ok"
#6
Go into the ".minecraft" --> "saves" folder and paste the copied folder (check that there is not another folder inside the folder. If this is so, the world might not be playable)

#2 Import the world into your Minecraft folder

#1
Right click on the .zip file "Projekt City [V5.0]" and click "extract all"
#2
Click on the bottom right on "extract"
#3
Wait for the file to extract and return to your previous directory
#4
Right click on the extracted file and copy it
#5
Go to your server folder
#6
Paste the copied folder in the root directory of the server
#7
Rename the folder if neccessary to "world" and if you have an existing "world" folder, remove the old folder. (When removing the old world folder, you are removing the world of your server which has been generated so far. Deleting this file can not be undone and you will lose any progress made in the world. Please keep that in mind!)
You can also add the world as a new world with the help of multiverse core! Apex Hosting has very good step-by-step instructions for this, available at the following address: https://apexminecrafthosting.com/multiverse-core-plugin/
